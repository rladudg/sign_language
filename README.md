청각장애인용 화상통화 보조 시스템

수화와 음성을 실시간으로 텍스트로 변환해 한 화면에 함께 표시하는 화상통화 보조 시스템입니다.
수화로 검출된 개별 자모(초·중·종성)는 한글 오토마타로 완성형 글자로 조합하고,
음성은 STT로 변환하여 청각장애인과 비장애인 간 양방향 의사소통을 지원합니다.


화상통화 영상은 기존 화상통화 솔루션 연동을 가정하며, 본 시스템은 그 위에 얹히는 실시간 자막 통합을 담당합니다.



<br>
주요 기능


수화 인식 결과로 들어온 개별 자모를 완성형 한글로 조합하는 한글 오토마타 직접 구현
멀티스레드 TCP 서버로 수화·음성 패킷을 실시간 수신
Python 음성 STT 입력과 수화 인식 입력을 하나의 화면에 통합
처리 상태(Ready / Detecting / Hold / Confirmed / Error) 표시 및 실시간 로그 UI


<br>
시스템 구조

[수화 인식 (팀원, Python)] ──┐
                            ├─→ TCP 패킷 ─→ [C# WinForms 서버] ─→ 자모 조합 / 자막 표시
[음성 STT (Python)]      ──┘


음성(Python): 마이크 입력을 Google STT로 텍스트 변환 후, 글자 단위로 VOICE:{글자} 패킷을 TCP로 전송
수화(팀원): 수화 인식 결과를 자모 단위 CHAR:{자모} 패킷으로 전송
서버(C#): 두 입력을 한 화면에서 수신·처리하며, 자모는 한글 오토마타로 완성형 글자로 조합


패킷 프로토콜은 텍스트 기반으로 단순하게 설계했습니다.

패킷의미CHAR:{자모}수화로 검출된 개별 자모 (오토마타로 조합)VOICE:{글자}음성 STT 변환 결과CMD:DELETE한 글자 삭제CMD:CLEAR전체 초기화STATE:{상태}처리 상태 표시

<br>
담당 역할 (통신 · 조합 · UI)


C# WinForms 멀티스레드 TCP 서버 구축 — 수화·음성 패킷을 별도 스레드에서 수신하고, Invoke로 UI 스레드에 안전하게 반영
한글 오토마타 직접 구현 — 개별 자모(초·중·종성)를 완성형 한글로 조합하는 로직을 라이브러리 없이 구현
Python 음성 STT 모듈 개발 — Google STT 기반 실시간 음성→텍스트 변환, 별도 스레드로 처리하고 인식 결과를 TCP로 전송
두 입력의 단일 화면 통합 — 수화·음성 입력을 한 화면에 합치고 상태 표시 및 실시간 로그 UI 구성



수화 인식 모델은 팀원이 담당했으며, 본 파트는 그 결과로 전달된 자모 패킷의 수신·조합·표시를 담당했습니다.
